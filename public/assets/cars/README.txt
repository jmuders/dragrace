Place the following car sprite PNG files in this folder:

  silver.png  — Image 1: Silver BMW M3-style sedan (player car 1)
  red.png     — Image 3: Red Audi R8-style supercar (player car 2)
  green.png   — Image 5: Green Lamborghini-style supercar (player car 3)
  white.png   — Image 4: White/silver BMW sedan (player car 4)
  orange.png  — Image 2: Orange supercar (CPU opponent)

The game will automatically use these sprites. If a file is missing,
the game falls back to the built-in procedural pixel art.
